Current Events Portal

For Your Consideration — Archive Process

Portal

jhnday.github.io/CrntEvnts/current_affairs_central.html

Archive

jhnday.github.io/CrntEvnts/FYC_Archive.html

Requirement

Both files live on GitHub Pages

Commands

Archive a single pill

Archive Pill/Reference: [date] | [pill text]

Claude fetches the live portal, finds the pill by text, extracts its URL and source card label, then inserts the entry into the archive under the correct quarter and date.

Archive from a screenshot

Archive Pill/Reference: [date]

Upload a screenshot of any number of pills alongside this command. Claude reads all pill texts from the image and archives them in one pass.

Archive and remove from portal

Archive Pill/Reference: [date] and remove from portal

Claude returns both files updated — the archive with the new entry, the portal with the pill removed. Upload both to GitHub.

Remove only

Remove from Portal: [pill text]

Removes the pill from the live portal without archiving it. Returns updated portal HTML.

What Claude does automatically

Fetches the live portal and archive from GitHub Pages

Extracts the URL from the pill's href — no need to supply it

Records the source card label as provenance in the archive

Files the entry in the correct quarter and date block

Your role

Step 1

Issue the command

Type or paste the command, optionally with a screenshot

Step 2

Download the files

Claude returns the updated HTML file(s)

Step 3

Upload to GitHub

Push to the repo — GitHub Pages serves the update live
